# Colorly

# 概要
Colorlyは、パーソナルカラーに特化したコスメ情報の共有プラットフォームです。より詳細に、自分に似合うコスメを見つけられます。  
また、顔タイプや骨格などファッション要素は取り入れず、あえてコスメのみに絞ることで、シンプルかつ使いやすい“イメコン”コミュニティを目指しました。  

---

# サイトテーマ
より迷わず、シンプルに自分をアップデートするヒントを。

---

# テーマを選んだ理由 
16タイプに分け、コスメに特化することで、より“私に似合う”が見つかる場を目指しました。

---

# きっかけ
似合う色やコスメが人それぞれ違う中で、欲しい情報にたどり着けない人が多いと感じました。  
SNSはコスメ以外の情報も混ざり、求める情報にたどり着きづらいため、コスメに特化した場を作りたいと思いました。  

---

# 問題点
・パーソナルカラーの細分化により、特定タイプの情報にアクセスしづらい。  
・SNS上ではコスメ・服・アクセサリーなどのジャンルが混在し、情報が煩雑。  
・自己診断とプロ診断が混ざっており、信頼性の判断が難しい。  

---

# 解決策
・ジャンルをコスメのみに絞ることで、検索性と明確さを向上。  
・投稿にはパーソナルカラータイプ（4・16タイプ）を紐づけて整理。  
・自己診断／プロ診断でフィルターがかけられる設計にし、情報の信頼性も明確に。  

---

# ターゲットユーザー
・16タイプ診断など詳細な属性で情報を探したい人。  
・「流行だから」ではなく、「自分に合う」を重視したい人。  
・情報過多のSNSに疲れ、自分に合う情報にたどり着けずにいる人。  

---

# 主な利用シーン
・新しいコスメを買う前に、自分のパーソナルカラーに合うかチェックしたいとき  
　→ タグや属性検索で、同じタイプの人のおすすめを参考にできる  
・流行りコスメが似合わず、モヤモヤしたときに「自分にはこれが似合う」と納得したいとき  
　→ 流行にとらわれず「似合う」基準で選ばれた情報が集まるから、自信を持って選べる  
・「これが似合う」と言われたコスメがしっくりこないとき、他の選択肢を探したいとき  
　→ 同じタイプでも好みや悩みによって選ばれたコスメ情報が見つかる
